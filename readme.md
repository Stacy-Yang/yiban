## 简单规范

### 后台

开发以应用为单位，控制器命名`AppName.php`，模型命名`AppName_Model.php`，插件放在`applications/third_party`目录下。应用生成或者上传文件目录设置在`./attch/appname`

### 前端

前端页面放置在`applications/views/appname`下，前端资源放置在`./pagestuff/appname`下。插件放在`./plugin`目录下
